**GIT Commands**

Check if Git is installed and see its version : **git --version**
Create a new Git repository in the current folder : **git init**
Check the current state of files (tracked, untracked, staged) : **git status**
Add a specific file to the staging area : **git add <filename>**
Add all new and modified files to the staging area : **git add .**
Save a snapshot of staged changes with a message : **git commit -m "Initial commit"**
View the history of commits made in the repository : **git log**
Usually the first file in a repository; explains what the project is about : **README.md**
