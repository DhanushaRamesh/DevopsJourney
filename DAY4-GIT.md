**GIT Commands**

1. Check if Git is installed and see its version : **git --version**
2. Create a new Git repository in the current folder : **git init**
3. Check the current state of files (tracked, untracked, staged) : **git status**
4. Add a specific file to the staging area : **git add <filename>**
5. Add all new and modified files to the staging area : **git add .**
6. Save a snapshot of staged changes with a message : **git commit -m "Initial commit"**
7. View the history of commits made in the repository : **git log**
8. Usually the first file in a repository; explains what the project is about : **README.md**
